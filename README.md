Howdy, this is your friendly neighborhood Cobblemon modpack!

The short guide:
1. Download all these mods and place them in your preferred mod launcher in a 1.21.1 instance (I recommend Prism launcher).
2. Make sure NeoForge is installed in your launcher. Newest version seems to work fine as of 5/22/2026.
3. Ensure you have both Cobblemon and Create Aeronautics added to the modlist as well, linked separately.
4. Play the game (or don't, who am I to judge).

Cobblemon:
https://modrinth.com/mod/cobblemon/version/1.7.3

Create Aeronautics:
https://modrinth.com/mod/create-aeronautics/version/YhZLrAFC

Happy modding!
